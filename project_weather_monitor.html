Overview

This project is a simple DIY smart weather monitor using an ESP32 and DHT11 sensor.

Parts List

  • ESP32 Dev Board
  • DHT11 Temperature/Humidity Sensor
  • OLED Display 0.96 inch
  • 3D Printed Case (download below)

3D Model

Download STL File

Instructions

  1. Connect the DHT11 sensor to ESP32 (GND, VCC, Data -> GPIO4).
  2. Connect the OLED display via I2C to ESP32.
  3. Flash the code below using Arduino IDE.
